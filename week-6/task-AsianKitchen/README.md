# Asian Kitchen Menu

Javascript pratik projesi: Asya mutfağı menüsünü dinamik olarak gösteren, kategorilere göre filtreleme yapan küçük bir uygulama.

## Özellikler
- Menü verileri JavaScript ile oluşturulur.
- Kategorilere göre butonlar otomatik üretilir.
- Tüm ürünler veya seçilen kategoriye ait ürünler listelenir.
- Menü kartları yerel görsellerle görüntülenir.

## Kurulum
1. Depoyu veya klasörü yerel makinenize alın.
2. `index.html` dosyasını herhangi bir modern tarayıcıyla açın.

## Proje Yapısı
```
index.html
css/
  style.css
img/
  
js/
  app.js
```

## Kullanım
- Sayfa açıldığında tüm menü elemanları listelenir.
- Üstteki kategori butonlarına tıklayarak filtreleme yapabilirsiniz.
