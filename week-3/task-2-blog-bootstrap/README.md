# Kamp Rehberi

Bu proje, Bootstrap 5 bileşenlerini kullanarak kart tabanlı bir blog ana sayfası ve basit bir blog gönderisi sayfası üzerinde hizalama ve tipografi pratikleri yapmayı amaçlar.

## İçerik

- Navbar, kart listesi ve sabit footer içeren ana sayfa (`index.html`)
- Kart düzeninden türetilmiş sade blog gönderisi (`post.html`)

## Nasıl Kullanılır?

1. Proje klasörünü yerel ortamınıza kopyalayın veya indirin.
2. `index.html` veya `post.html` dosyalarını tarayıcıda açın.
3. Bootstrap 5 CDN üzerinden yüklendiğinden ek kurulum gerekmez.

## Hedeflenen Bootstrap Pratikleri

- Grid sistemi (`row`, `col`, responsive kırılımlar)
- Kart bileşenleri ve yardımcı sınıflar (`p`, `mt`, `shadow`, `rounded`)
- Navbar hizalama ve responsive davranış
- Footer’ı sayfa altında sabitleme ve esnek düzen kullanımı
